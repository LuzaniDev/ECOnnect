# v1.3.0 — Geração Inteligente de Código de Barras + PDF Watcher + Calculadora

## Novas Funcionalidades

### 🧾 Extração de Código de Barras via PDF
- Criado `boleto_pdf.py` — serviço que localiza o PDF do boleto em `DIRETORIOGERACAOBOLETO` e extrai a linha digitável usando PyMuPDF
- Criado `boleto_watcher.py` — monitor em background que varre os diretórios de boletos configurados e alimenta automaticamente a tabela `BOLETO_GERADO` no Firebird
- Criada tabela `BOLETO_GERADO` — armazena código de barras, linha digitável e caminho do PDF vinculado à parcela
- `_calcular_barcode` agora consulta `BOLETO_GERADO` primeiro — sem cálculo manual, sem SQLite, dados reais do PDF do eCobranca

### 🧮 Calculadora Animada
- Nova `calculadora.py` com dialog estilizado dark mode
- Botões com efeito _bounce_ via `QPropertyAnimation`
- Ícones SVG para todas as operações (backspace, clear, divisão, multiplicação, etc.)
- Botão de acesso na barra de ferramentas ao lado do "Visualizar"

### 🎨 UI / UX
- **Coluna "Banco"** adicionada ao Ver Boletos (Sicredi, BB, Sicoob, etc.)
- **Status dinâmico**: boletos com código de barras em `BOLETO_GERADO` exibem "Registrado" automaticamente
- **Botão "Copiar"** com ícone SVG e padding adequado (sem corte)
- **Calculadora** com animação de pressionar/soltar

## Correções

### 🐞 Bug de deduplicação de parcelas
- Corrigido `_on_result` que descartava a 2ª parcela do mesmo cliente porque deduplicava por `(empresa, cliente)`
- Agora todas as parcelas são listadas individualmente

### 🐛 Ícone SVG via QIcon.addData
- Substituído `QIcon.addData()` por `QImage.fromData()` + `QPixmap` — compatível com PySide6

### 🔧 Filtro "Apenas boletos registrados" removido
- Substituído pelo status automático "Registrado" com base em `BOLETO_GERADO`

### 🛠️ Código de Barras (barcode.py)
- **Sicredi (748)**: campo livre corrigido para `11 + NN(9) + AG(4) + POSTO(2) + CODIGOCEDENTE(5) + 10 + DV`
- **Banco do Brasil (001)**: campo livre corrigido para `000000 + NN(17) + CARTEIRA(2)`
- **Fator**: ajuste `-9000` quando dias > 9000 (compatível com eCobranca)
- Validação cruzada com boleto real ALIANÇA R$48,00 ✅

### 🔗 Dependências
- Adicionado `PyMuPDF>=1.28.0` ao `requirements.txt`

- Watcher usa `FirebirdClient` dinâmico (lê `.env`)
- Paths de rede (UNC) ignorados para não travar

### 📋 Placeholders
