# v1.3.1 — Graphify + Boleto Watcher + Data Pipeline

**Data**: 2026-07-06

## Novidades

### Grafo de Conhecimento (Graphify)
- Instalado e configurado Graphify para geracao de grafo de conhecimento do codigo
- Skill graphify-auto-update para OpenCode (reconstroi grafo a cada mudanca)
- Git hooks (post-commit, post-checkout) para rebuild automatico
- 1060+ nos, 2694+ arestas, 76+ comunidades mapeadas

### Data Pipeline 3D
- Dashboard interativo 3D (Canvas 2D puro, sem WebGL) mostrando tabelas Firebird, PostgreSQL, endpoints API e codigo fonte
- Acessivel em Administracao > Data Pipeline
- Busca, filtro por grupo, zoom, tooltip

### Boleto Watcher (watchdog)
- Substituido polling de 30s por watchdog (deteccao instantanea de novos PDFs)
- Auto-criacao da tabela BOLETO_GERADO se nao existir
- Suporte a paths UNC (`\\servidor\...`)
- Scan inicial com overlay de progresso no pos-login

### Configuracoes
- Nova secao "Diretorios de Boletos" nas Configuracoes do app

## Correcoes

### Boleto Watcher
- Regex do `_parse_filename` corrigido para capturar `num_boleto` com underscore interno
- `_processar_pdf` agora retorna bool (contagem precisa de boletos processados)
- Threading.Lock no FirebirdClient para evitar deadlock entre threads
- Scan inicial roda inline (sem thread) para evitar conflito de conexao Firebird
- Conexao fdb dedicada no filtro de cobranca (elimina deadlock Firebird embedded)
- PyMuPDF instalado no .venv (faltava)

### Filtro Cobranca
- COBRANCA_SQL simplificada: removidos LEFT JOINs lentos (TRecParametro, TRECBOLETO), calculos movidos para Python
- Query caiu de 120s+ para ~0.05s
- Parametros de paginacao reposicionados (FIRST/SKIP antes dos WHERE)
- `_calcular_barcode` agora usa IDTRECPARCELA em vez de NOSSONUMERO (coluna removida na SQL simplificada)

### Dashboard
- Corrigido `NameError: name 't' is not defined` em `_render_activity()` e `_render_runs()`

### Interface
- Menu Tipo Pessoa / Tipo Cliente: substituido QMenu com _CheckableMenu por popup QDialog + QCheckBox
- Botao "Ver Boletos" renomeado para "Ver Pendencias" com query direta em TRecParcela
- Dialog de pendencias agora mostra Status (Vencida, A vencer, Paga, etc.) e Tipo (nome da carteira)
- Botao "12 meses" nos filtros de data
