# ECOnnect v1.0.0

**Data**: —

## Novidades
- Versão inicial do ECOnnect
- Interface desktop com PySide6
- API FastASYNC com gerenciamento de integrações
- Autenticação via ECO Firebird + JWT
- Dashboard com métricas de execução
- Templates de requisição reutilizáveis com variáveis
- Testador de requisições em tempo real
- Agendamento de integrações (cron)
- Auditoria e histórico de execuções
- Suporte a múltiplas empresas e usuários
- Tema escuro customizado

## Dependências
- Python 3.10+
- PostgreSQL 12+
- Firebird 2.5+ (opcional)
- Node.js 18+ (apenas para build do instalador)
