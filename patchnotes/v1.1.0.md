# ECOnnect v1.1.0

**Data**: 2026-06-11

## Novidades

### Módulo Mundo dos Bots
- Aba completa "Mundo dos Bots" com integração ao sistema de mensagens
- Templates de mensagem reutilizáveis com suporte a **tags** (exibidas como `nome [tag]`)
- Envio de mensagens em lote com fila de processamento e verificação de cooldown
- Agendamento de jobs com execução em background e barra de progresso
- Aba **Cobrança** com filtros por período (data inicial/final), tabela de clientes com status e coluna "Cód. Cliente"
- Indicadores de status com cores: Nenhum (cinza), Carregando (laranja), Carregado (verde), com falha (vermelho)
- Tabela de jobs com colunas redimensionadas, largura mínima, e coluna "Ações" com botão "Ver" (exibe clientes do job)
- Diálogo de clientes ocultos com opção de remoção (botão "Remover")

### Sistema de Cooldown por Tag
- Configuração individual de bloqueio (em horas) por tag via página de Configurações
- Armazenamento persistente em `~/.econnect/tag_cooldown_config.json`
- Verificação automática de cooldown no envio de mensagens e execução de jobs
- Histórico de envios em `~/.econnect/sent_history.json` para controle de tempo

### Janela Frameless
- Janela principal sem moldura padrão do Windows, com bordas arredondadas (16px)
- Barra de título customizada com ícone e botão de fechar
- Redimensionamento por arrasto nas bordas (margem de 8px) via `WM_NCHITTEST`
- Arrasto da janela pela barra de título
- Atalho **Win+↑** para maximizar e **Win+↓** para restaurar/minimizar (via low-level keyboard hook)

### Aba de Requisições
- View dedicada para requisições com integração ao banco Firebird
- Suporte a variáveis SQL e agendamento

### Abas de Administração
- Gerenciamento de usuários com permissões
- Tabs administrativas para configuração do sistema

## Melhorias
- Coluna "Cód. Cliente" adicionada na tabela de cobrança (índice 1)
- Labels de status da cobrança com cores distintas para cada estado
- Serialização robusta de objetos `date`/`datetime`/`bytes` antes de `json.dump`
- Botão "Filtrar" alinhado à direita na barra de filtros
- Altura de linhas ajustada para 44px no diálogo de clientes ocultos com coluna de ação em 150px
- Combos de template exibem `nome [tag]` para facilitar identificação
- Configuração de cooldown por tag migrada de `QFormLayout` para `QTableWidget` com visualização clara
- QSpinBox do cooldown ajustado para 22px de altura (proporcional à linha da tabela)
- Persistência de configurações Firebird via API dedicada (`company_config_api`)

## Correções
- **Serialização**: Erro ao salvar histórico de envios com objetos `date`/`datetime` — adicionado `_serialize_row`
- **Layout cooldown**: Tags não apareciam na listagem; substituído layout customizado por `QTableWidget`
- **Alinhamento**: Botão "Filtrar" estava deslocado horizontalmente
- **Altura spinbox**: Dimensão do QSpinBox excedia a altura da linha na tabela de cooldown
- **Template path**: Unificado caminho do arquivo de templates entre settings e mundo_bots

## Procedimentos de Atualização
1. Nenhuma migração manual necessária
2. Arquivos de configuração locais serão criados automaticamente em `~/.econnect/`:
   - `mundo_bots_templates.json` — templates do Mundo dos Bots
   - `tag_cooldown_config.json` — configuração de cooldown por tag
   - `sent_history.json` — histórico de envios
3. A versão do VERSION file foi atualizada de 1.0.0 para 1.0.1 (preparatório)

## Dependências
- Python 3.10+
- PySide6 (Qt for Python)
- FastAPI + Uvicorn
- Firebird 2.5+ (opcional, para integrações)
