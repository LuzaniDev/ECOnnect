# ECOnnect v1.2.2

**Data**: 2026-06-15

## Novidades

### Integração com IA (Ollama)
- Novo tipo de requisição **IA** — envia payload para API compatível com Ollama (`/api/generate`)
- Resposta renderizada em Markdown com `AIResponseDialog` (títulos, listas, links, negrito)
- Se o payload tiver `"stream": true`, usa **streaming SSE** com tokens em tempo real via `StreamDialog`
- Streaming não-streaming: fallback para requisição normal com resposta formatada

### Integração com n8n
- Novo tipo de requisição **n8n** — envia webhook para URL configurada com headers personalizados
- `N8nLoadingDialog` com barra de progresso indeterminada e botão Cancelar
- `N8nResponseDialog` exibe `base_conhecimento[]` e `uar_solicitacoes[]` em cards com links clicáveis
- Timeout de 1200s (20 minutos) para respostas lentas
- Quando estrutura esperada não é encontrada, exibe o JSON bruto como fallback

### Auto-fill de URL por Tipo
- Ao selecionar tipo **IA**, URL é preenchida com `http://192.168.1.94:11434/api/generate`
- Ao selecionar tipo **n8n**, URL é preenchida com `http://seu-n8n.com/webhook/`

## Melhorias

### Diálogo de Edição
- Combobox de tipo agora inclui **IA** e **n8n** (além de Normal e Cobranca)
- Tipo salvo é restaurado corretamente ao editar

### Tabela de Requisições
- Coluna "Tipo" exibe **IA** ou **n8n** com cor `accent_blue`

### Fluxo de Execução
- Requisições do tipo IA não enviam headers customizados no trigger (apenas `Content-Type`)
- Requisições do tipo n8n ignoram `X-ACCESS-TOKEN` (usa headers manuais)

## Correções

- **HTML entities no n8n**: títulos com `&nbsp;`, `&amp;`, tags HTML são decodificados via `html.unescape()` + regex de remoção de tags
- **Trigger com variáveis**: payload substituído é passado como `override_payload` corretamente em todos os tipos
- **JSON inválido**: validação de JSON ocorre antes de qualquer requisição, incluindo após substituição de variáveis

## Técnico

- Backend: novo endpoint `POST /{config_id}/trigger-stream` (SSE) no router de integrações
- Backend: branch `elif config.type == "ia"` e `elif config.type == "n8n"` no `IntegrationService.trigger()`
- Frontend: classes `AIResponseDialog`, `N8nResponseDialog`, `N8nWorker`, `N8nLoadingDialog`, `StreamWorker`, `StreamDialog`
- Frontend: método `stream_post()` no `ApiClient` para SSE
- Frontend: `_on_create_type_changed` conectado ao `currentTextChanged` do combo de criação
