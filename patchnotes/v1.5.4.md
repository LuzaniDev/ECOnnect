# v1.5.4 - Edicao Geral, Status Unificado e Pydantic V2

**Data**: 2026-07-11

## Novidades

### Edicao de todos os campos (nao apenas vazios)
- "Editar Campos" agora permite editar qualquer campo, independentemente de estar vazio
- Valores atuais sao pre-carregados no dialogo para facilitar a edicao
- Campo "Nome" adicionado aos campos editaveis
- Feedback melhora: mostra quantos clientes foram salvos ou "Nada alterado" se nenhuma modificacao ocorreu

### Coluna "Dias Atraso" removida (fundida com Status)
- Coluna "Dias Atraso" removida da tabela principal para simplificar o layout
- A informacao de atraso agora aparece diretamente no texto do Status
- Ex: "Vencida 5 dia(s)", "A vencer 3 dia(s)", "Vence hoje"

### Status "Ja Enviado" mais robusto
- Fallback para verificacao local de cooldown quando o cache remoto esta vazio
- Consulta o historico de envios local como ultimo recurso
- Garante que o status "Ja Enviado" seja preenchido corretamente mesmo sem cache

### Ver Pendencias simplificado
- Coluna "Dias" removida da tabela de pendencias (informacao agora no Status)
- Layout mais limpo e objetivo

## Correcoes

### Pydantic V2 migration
- `backend/app/config.py`: `Config` class interna substituida por `model_config = SettingsConfigDict(...)`
- Corrige warning de deprecacao e garante compatibilidade com Pydantic V2
- `extra = "ignore"` movido para o local correto (dentro de `model_config`)

### PyInstaller frozen build
- `ECOnnect.spec`: Adicionado `rthook_shiboken.py` como runtime hook
- Excluido `shibokensupport` dos imports para evitar erro em executavel frozen
- Corrige falha ao executar o binario compilado

### Atualizacao do icone do app
- `frontend/assets/app_icon.ico` atualizado
